# NIDS Incremental Feature Extraction Project

## Overview
This project implements incremental feature extraction using recurrence relations for use in Network Intrusion Detection Systems (NIDS).

The system processes network packets using sliding windows and computes statistical features such as running mean and variance.

## Features
- Incremental mean calculation
- Running variance using Welford's algorithm
- Sliding window packet processing
- Real-time feature extraction
- Efficient memory usage
